# Experiments

| Directory | Target |
|-----------|--------|
| `simple/` | Basic checkerboard and stripe pattern comparisons across evolution methods |
| `local_patterns/` | Local pattern generation (checkerboard, diagonal-bands, dense-dots) with truth table extraction and analysis |
| `complex_patterns/` | More complex target pattern generation |
| `generalization/` | Testing whether evolved rules generalize across different grid sizes/conditions |
| `shared/` | Common Python analysis utilities used across experiments |
